## How to run the scripts

First make sure you are in the right directory and are in a virtual environment.

Run each script in this order:

download_data.sh
analyze_logs.sh
generate_report.sh
run_pipeline.sh


Before running each script, input the following in your terminal:

chmod +x script_name.sh

Then after doing that, run this:
bash script_name.sh

So for example, to run download_data.sh, do this in your terminal:
chmod +x download_data.sh
bash download_data.sh

My terminal would look like this:
(venv) kathymo@Kathys-MacBook-Pro kathymo % chmod +x download_data.sh
(venv) kathymo@Kathys-MacBook-Pro kathymo % bash download_data.sh

You repeat this for all the other scripts. If you want to run analyze_logs.sh, it would be:
chmod +x analyze_logs.sh
bash analyze_logs.sh
